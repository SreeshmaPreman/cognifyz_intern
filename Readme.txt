# Nova Nights - Web Development Internship Project

## 👩‍💻 Internship Level:
Cognifyz Web Development Internship - Level 3 (Landing Page & Image Gallery)

## 🌟 Description:
Nova Nights is a multi-section event-themed website designed for a web development internship. It includes:
- A beautiful landing page with event countdown
- An image gallery showcasing the event
- Details about the organizers
- A contact form (optional page)

## 🛠️ Technologies Used:
- HTML5
- CSS3 (with Bootstrap)
- JavaScript (for countdown)

## 📁 Folder Structure:
- index.html
- style.css
- organized-by.css
- images/
- video.mp4
- README.txt

## 🚀 How to Run:
1. Download the ZIP file and extract it.
2. Open `index.html` in any web browser.
